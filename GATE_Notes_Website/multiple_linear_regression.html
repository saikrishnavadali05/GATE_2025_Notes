Introduction to Multiple Linear Regression

Multiple linear regression is an extension of simple linear regression. It is used to predict the value of a dependent variable based on the values of two or more independent variables.

How Multiple Linear Regression Works

The multiple linear regression model assumes a linear relationship between the dependent variable and each of the independent variables. The general form of the multiple linear regression equation is:

y = b0 + b1*x1 + b2*x2 + ... + bn*xn

Where:

  • y is the dependent variable
  • b0 is the y-intercept
  • b1, b2, ..., bn are the coefficients of the independent variables x1, x2, ..., xn

Assumptions of Multiple Linear Regression

The multiple linear regression model makes several key assumptions:

  • There is a linear relationship between the dependent variable and the independent variables.
  • The residuals (errors) are normally distributed.
  • There is little or no multicollinearity in the data (independent variables are not highly correlated).
  • Homoscedasticity: The variance of the residuals is constant across all levels of the independent variables.
  • The observations are independent of each other.

Example in Python

Here is a simple example of how to perform multiple linear regression using Python and the scikit-learn library:

import numpy as np
from sklearn.linear_model import LinearRegression

# Example data
X = np.array([[1, 1], [1, 2], [2, 2], [2, 3]])
y = np.dot(X, np.array([1, 2])) + 3

# Create a multiple linear regression model
model = LinearRegression().fit(X, y)

# Make predictions
predictions = model.predict(X)

# Output results
print("Coefficients:", model.coef_)
print("Intercept:", model.intercept_)
print("Predictions:", predictions)

Conclusion

Multiple linear regression is a powerful tool for understanding the relationship between multiple independent variables and a dependent variable. It is widely used in various fields, including finance, economics, biology, and social sciences.
