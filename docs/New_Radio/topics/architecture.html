5G NR Architecture

5G architecture is designed to be flexible, scalable, and service-oriented. It enables support for eMBB, URLLC, and mMTC use cases through a modular and virtualized network structure.

The core components include the gNB (Next-Gen Node B or base station), 5GC (5G Core), and UE (User Equipment).

  • Standalone (SA): Uses 5G RAN and 5G Core. It is fully independent from 4G.
  • Non-Standalone (NSA): Uses 5G RAN with existing 4G Core (EPC). It allows fast initial deployment.
  • UE: User equipment like smartphones and IoT devices
  • gNB: 5G base station connected to the 5G core
  • AMF: Access and Mobility Management Function
  • SMF: Session Management Function
  • UPF: User Plane Function handles data forwarding
  • AUSF: Authentication Server Function
  • PCF: Policy Control Function

The 5G Core is built on a Service-Based Architecture where all network functions (NFs) interact using REST APIs over HTTP/2. This improves flexibility, scalability, and network slicing capabilities.
