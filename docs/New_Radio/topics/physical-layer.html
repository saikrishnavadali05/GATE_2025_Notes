5G NR Physical Layer

The physical layer (PHY) is responsible for actual transmission and reception of data over the air interface. It handles modulation, coding, resource mapping, and more.

  • OFDMA: Used for downlink and uplink
  • Massive MIMO: Enhances throughput and coverage using multiple antennas
  • Beamforming: Directs signal energy to specific directions
  • Dynamic TDD: Adjusts uplink and downlink slots dynamically
  • Flexible Numerology: Multiple subcarrier spacings (15, 30, 60, 120, 240 kHz)

5G supports scalable subcarrier spacing, referred to as numerology μ:

  • μ = 0 → 15 kHz (same as LTE)
  • μ = 1 → 30 kHz
  • μ = 2 → 60 kHz
  • μ = 3 → 120 kHz
  • μ = 4 → 240 kHz (used for mmWave)

5G uses 10 ms frames, each divided into 1 ms subframes. The number of slots per subframe depends on the numerology. For example:

  • 15 kHz → 1 slot/subframe
  • 30 kHz → 2 slots/subframe
  • 120 kHz → 8 slots/subframe
  • PDSCH: Physical Downlink Shared Channel
  • PUSCH: Physical Uplink Shared Channel
  • PDCCH: Downlink Control Channel
  • PUCCH: Uplink Control Channel
  • PBCH: Broadcast Channel
