5G NR Protocol Stack

The 5G NR protocol stack is divided into two planes:

  • User Plane: Handles data traffic (e.g., video, web)
  • Control Plane: Manages signaling and control messages
  • PDCP (Packet Data Convergence Protocol): Header compression, encryption
  • RLC (Radio Link Control): Segmentation, retransmission
  • MAC (Medium Access Control): Scheduling, HARQ
  • PHY (Physical Layer): Modulation, channel coding
  • RRC (Radio Resource Control): Connection setup, mobility, security
  • PDCP, RLC, MAC, PHY: Same layers as user plane with control-specific functions

Both gNB (base station) and UE (user equipment) implement the same protocol layers. Communication happens layer-to-layer between them (e.g., PDCP of UE ↔ PDCP of gNB).

Here's a simplified view of the 5G NR stack:

    Application
    └── NAS (Non-Access Stratum)
        └── RRC
            └── PDCP
                └── RLC
