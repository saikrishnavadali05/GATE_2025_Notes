2. Initial Access in 5G NR

Initial Access is the first step a mobile device (UE) performs to connect with a 5G NR base station (gNB). It includes cell search, synchronization, and random access procedures.

  • UE looks for Synchronization Signal Blocks (SSBs) broadcast by gNB.
  • Each SSB contains:
    • PSS: Primary Synchronization Signal
    • SSS: Secondary Synchronization Signal
    • PBCH: Physical Broadcast Channel
  • The UE uses PSS/SSS to detect the cell and synchronize frequency/time.

After synchronization, the UE reads the PBCH to obtain basic configuration. Additional information is fetched from:

  • SIB1: Contains cell configuration needed for access
  • Other SIBs: May be broadcast on-demand or periodically

The UE uses the RACH to request a dedicated connection. Two types exist:

  • Contention-Based RACH: Multiple UEs may try simultaneously (requires resolution)
  • Contention-Free RACH: gNB assigns resources (used in handover or special cases)
  1. UE sends a random access preamble
  2. gNB replies with a Random Access Response (RAR)
  3. UE sends RRC Connection Request
  4. gNB responds with RRC Connection Setup
  5. UE completes the process with RRC Connection Setup Complete

Since 5G uses directional antennas (beamforming), SSBs are transmitted in different directions using beam sweeping. The UE selects the best beam for communication.
