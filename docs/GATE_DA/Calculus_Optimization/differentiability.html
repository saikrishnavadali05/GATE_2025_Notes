Differentiability

Back to Calculus and Optimization Topics

Differentiability is a fundamental concept in calculus that describes the existence of a derivative for a function at a given point. If a function is differentiable at a point, it means that it has a well-defined tangent at that point, and the function can be locally approximated by a linear function.

Definition

A function \( f(x) \) is said to be differentiable at a point \( x = a \) if the following limit exists:

\[ f'(a) = \lim_{{h \to 0}} \frac{{f(a + h) - f(a)}}{h} \]

If \( f(x) \) is differentiable at every point in an interval, then it is said to be differentiable on that interval.

Examples

Differentiable Function

Consider the function \( f(x) = x^2 \). To check if it is differentiable at \( x = a \), we calculate the limit:

\[ f'(a) = \lim_{{h \to 0}} \frac{{(a + h)^2 - a^2}}{h} = \lim_{{h \to 0}} \frac{{a^2 + 2ah + h^2 - a^2}}{h} = \lim_{{h \to 0}} \frac{{2ah + h^2}}{h} = \lim_{{h \to 0}} (2a + h) = 2a \]

Therefore, \( f(x) = x^2 \) is differentiable at \( x = a \) with \( f'(a) = 2a \).

Non-Differentiable Function

Consider the function \( g(x) = |x| \). To check if it is differentiable at \( x = 0 \), we calculate the left-hand and right-hand limits:

\[ g'(0^-) = \lim_{{h \to 0^-}} \frac{{|0 + h| - |0|}}{h} = \lim_{{h \to 0^-}} \frac{{-h}}{h} = -1 \]

\[ g'(0^+) = \lim_{{h \to 0^+}} \frac{{|0 + h| - |0|}}{h} = \lim_{{h \to 0^+}} \frac{{h}}{h} = 1 \]

Since the left-hand and right-hand limits are not equal, \( g(x) = |x| \) is not differentiable at \( x = 0 \).

Relationship Between Continuity and Differentiability

  • If a function is differentiable at a point, it is also continuous at that point.
  • If a function is not continuous at a point, it is not differentiable at that point.
  • However, a function can be continuous at a point but not differentiable at that point (e.g., \( g(x) = |x| \) at \( x = 0 \)).

Higher-Order Derivatives

If \( f(x) \) is differentiable, its derivative \( f'(x) \) may also be differentiable. If \( f'(x) \) is differentiable, its derivative is called the second-order derivative of \( f(x) \), denoted as \( f''(x) \). This process can continue, leading to higher-order derivatives \( f^{(n)}(x) \) for \( n \geq 1 \).

Applications

  • Physics: Used to describe motion, forces, and other physical phenomena.
  • Engineering: Essential for analyzing and designing systems and structures.
  • Economics: Helps in modeling and analyzing changes in economic variables.
  • Biology: Used in modeling growth rates and other dynamic processes.
  • Computer Science: Applied in algorithms and optimization problems.

Conclusion

Differentiability is a key concept in calculus that extends the idea of continuity by describing how a function changes at a given point. Understanding differentiability is essential for studying and applying calculus to real-world problems.
