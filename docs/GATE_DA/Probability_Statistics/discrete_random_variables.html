Discrete Random Variables

Back to Probability and Statistics Topics

A discrete random variable is a type of random variable that can take on a countable number of distinct values. These variables are used to model scenarios where outcomes are distinct and separated from each other.

Definition

A random variable \(X\) is discrete if it can take on only a finite or countably infinite number of distinct values. The probability mass function (PMF) of a discrete random variable \(X\) is a function \(p(x)\) that gives the probability that \(X\) equals \(x\):

\[ P(X = x) = p(x) \]

The PMF must satisfy the following properties:

  • \( 0 \leq p(x) \leq 1 \) for all \( x \)
  • \( \sum_{x} p(x) = 1 \)

Examples of Discrete Random Variables

Example 1: Binomial Random Variable

A binomial random variable counts the number of successes in a fixed number of independent Bernoulli trials, each with the same probability of success \( p \). The PMF of a binomial random variable \( X \) with parameters \( n \) and \( p \) is given by:

\[ P(X = k) = \binom{n}{k} p^k (1 - p)^{n - k} \]

where \( k = 0, 1, 2, \ldots, n \).

Example 2: Poisson Random Variable

A Poisson random variable counts the number of events occurring within a fixed interval of time or space, given that these events occur with a known constant mean rate \( \lambda \) and independently of the time since the last event. The PMF of a Poisson random variable \( X \) is given by:

\[ P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!} \]

where \( k = 0, 1, 2, \ldots \).

Expected Value and Variance

The expected value (mean) of a discrete random variable \( X \) with PMF \( p(x) \) is given by:

\[ E(X) = \sum_{x} x p(x) \]

The variance of \( X \) is a measure of the spread of its distribution and is given by:

\[ \text{Var}(X) = E[(X - E(X))^2] = \sum_{x} (x - E(X))^2 p(x) \]

Properties of Discrete Random Variables

  • Countable Outcomes: Can take on a finite or countably infinite number of distinct values.
  • Probability Mass Function (PMF): Describes the probability of each possible value.
  • Expected Value: The mean or average value of the distribution.
  • Variance: Measures the spread or dispersion of the distribution.

Applications

  • Quality Control: Used to model the number of defective items in a batch.
  • Healthcare: Used to model the number of occurrences of diseases or health conditions.
  • Insurance: Used to model the number of claims or occurrences of events.
  • Operations Research: Applied in queuing theory and inventory management.
  • Ecology: Used to model the count of species or individuals in a given area.

Conclusion

Discrete random variables are fundamental in probability and statistics, providing a framework for modeling and analyzing scenarios where outcomes are distinct and countable. Understanding their properties and applications is crucial for various fields, including quality control, healthcare, insurance, operations research, and ecology.
