Confidence Interval

Back to Probability and Statistics Topics

A confidence interval is a range of values, derived from sample data, that is likely to contain the true population parameter with a specified level of confidence. It provides an estimate of the parameter along with a measure of uncertainty.

Definition

A confidence interval for a population mean \( \mu \) is given by:

\[ \text{Confidence Interval} = \overline{X} \pm Z \frac{\sigma}{\sqrt{n}} \]

where:

  • \( \overline{X} \) = sample mean
  • \( Z \) = Z-score corresponding to the desired confidence level
  • \( \sigma \) = population standard deviation (or sample standard deviation if \( \sigma \) is unknown)
  • \( n \) = sample size

Confidence Levels

Common confidence levels include 90%, 95%, and 99%. The corresponding Z-scores for these confidence levels are approximately:

  • 90%: \( Z = 1.645 \)
  • 95%: \( Z = 1.96 \)
  • 99%: \( Z = 2.576 \)

Examples

Example 1: Population Mean

Suppose we have a sample of 50 students with a mean test score of 75 and a standard deviation of 10. We want to calculate the 95% confidence interval for the true mean test score.

Sample mean (\( \overline{X} \)) = 75
Sample standard deviation (\( s \)) = 10
Sample size (\( n \)) = 50
Z-score for 95% confidence (\( Z \)) = 1.96

Confidence Interval = 75 ± 1.96 * (10 / √50)
                   = 75 ± 1.96 * 1.414
                   = 75 ± 2.77
                   = (72.23, 77.77)

So, the 95% confidence interval for the true mean test score is (72.23, 77.77).

Example 2: Proportion

Suppose we have a sample of 200 voters, and 120 of them favor a particular candidate. We want to calculate the 95% confidence interval for the true proportion of voters favoring the candidate.

Sample proportion (\( \hat{p} \)) = 120 / 200 = 0.60
Sample size (\( n \)) = 200
Z-score for 95% confidence (\( Z \)) = 1.96

Standard error (\( SE \)) = √[ (0.60 * (1 - 0.60)) / 200 ]
                         = √[ 0.24 / 200 ]
                         = √0.0012
                         = 0.0346

Confidence Interval = 0.60 ± 1.96 * 0.0346
                   = 0.60 ± 0.0678
                   = (0.5322, 0.6678)

So, the 95% confidence interval for the true proportion of voters favoring the candidate is (0.5322, 0.6678).

Interpretation

A 95% confidence interval means that if we were to take 100 different samples and compute a confidence interval for each sample, we would expect approximately 95 of the intervals to contain the true population parameter.

Factors Affecting Confidence Intervals

  • Sample Size: Larger sample sizes result in narrower confidence intervals, indicating more precise estimates.
  • Variability: Higher variability in the data results in wider confidence intervals.
  • Confidence Level: Higher confidence levels result in wider confidence intervals.

Applications

  • Survey Analysis: Used to estimate population parameters based on sample data.
  • Quality Control: Used to determine the precision of process measurements and ensure they are within acceptable limits.
  • Clinical Trials: Used to estimate the effects of treatments and interventions.

Conclusion

Confidence intervals are a powerful tool in statistics, providing a range within which we can be reasonably sure the true population parameter lies. By understanding how to calculate and interpret confidence intervals, researchers and analysts can make more informed decisions based on sample data.
