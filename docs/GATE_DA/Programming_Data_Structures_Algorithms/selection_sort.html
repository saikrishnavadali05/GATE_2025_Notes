Selection Sort

Back to Programming, Data Structures, and Algorithms Topics

Selection sort is a simple comparison-based sorting algorithm. It works by repeatedly finding the minimum element from the unsorted part of the array and putting it at the beginning. The algorithm maintains two subarrays in a given array:

  • Sorted subarray: Initially empty and gradually grows to include elements in the correct order.
  • Unsorted subarray: Initially contains all the elements and shrinks as elements are moved to the sorted subarray.

Algorithm

The selection sort algorithm operates as follows:

  1. Start with the first element in the array.
  2. Find the minimum element in the unsorted part of the array.
  3. Swap the minimum element with the first unsorted element.
  4. Move the boundary between the sorted and unsorted subarrays one position to the right.
  5. Repeat steps 2-4 until the entire array is sorted.

Example

Consider the array \([64, 25, 12, 22, 11]\). The selection sort algorithm proceeds as follows:

  1. Find the minimum element (11) and swap it with the first element: \([11, 25, 12, 22, 64]\).
  2. Find the minimum element in the remaining unsorted part (12) and swap it with the second element: \([11, 12, 25, 22, 64]\).
  3. Find the minimum element in the remaining unsorted part (22) and swap it with the third element: \([11, 12, 22, 25, 64]\).
  4. Find the minimum element in the remaining unsorted part (25) and swap it with the fourth element: \([11, 12, 22, 25, 64]\).

After all iterations, the array is sorted.

Python Implementation

def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_idx = i
        for j in range(i+1, n):
            if arr[j] < arr[min_idx]:
                min_idx = j
        arr[i], arr[min_idx] = arr[min_idx], arr[i]

# Example usage
arr = [64, 25, 12, 22, 11]
selection_sort(arr)
print('Sorted array:', arr)

Time Complexity

The time complexity of selection sort is \( O(n^2) \) for all cases (worst, average, and best). This is because there are two nested loops, each of which runs approximately \( n \) times.

Space Complexity

The space complexity of selection sort is \( O(1) \), as it only requires a constant amount of additional memory space for the swapping operation.

Applications

  • Small Datasets: Selection sort is suitable for small datasets where the simplicity of the algorithm outweighs its inefficiency.
  • Educational Purposes: It is often used in teaching because it is easy to understand and implement.
  • Memory-Constrained Environments: Since it has a space complexity of \( O(1) \), it is useful in environments with limited memory.

Conclusion

Selection sort is a straightforward sorting algorithm that is easy to implement and understand. While it is not the most efficient sorting algorithm for large datasets, it is useful for small datasets and educational purposes.
