Introduction to Graph Theory

Back to Programming, Data Structures, and Algorithms Topics

Graph theory is the study of graphs, which are mathematical structures used to model pairwise relations between objects. A graph is composed of vertices (or nodes) and edges (or arcs) that connect pairs of vertices.

Basic Terminology

  • Vertex (Node): A fundamental unit of a graph, represented as a point.
  • Edge (Arc): A connection between two vertices, represented as a line.
  • Degree: The number of edges incident to a vertex. For a vertex \( v \), its degree is denoted by \( \deg(v) \).
  • Path: A sequence of vertices connected by edges, where each vertex is visited only once.
  • Cycle: A path that starts and ends at the same vertex, with no other repetitions.
  • Connected Graph: A graph in which there is a path between every pair of vertices.
  • Subgraph: A graph formed from a subset of the vertices and edges of a larger graph.

Types of Graphs

  • Undirected Graph: A graph in which edges have no direction. If there is an edge between vertices \( u \) and \( v \), it is denoted as \( \{u, v\} \).
  • Directed Graph (Digraph): A graph in which edges have a direction. If there is an edge from vertex \( u \) to vertex \( v \), it is denoted as \( (u, v) \).
  • Weighted Graph: A graph in which each edge is assigned a weight (or cost). The weight of an edge \( \{u, v\} \) is denoted as \( w(u, v) \).
  • Tree: A connected acyclic graph.
  • Bipartite Graph: A graph whose vertices can be divided into two disjoint sets such that no two vertices within the same set are adjacent.

Graph Representation

Graphs can be represented in various ways:

  • Adjacency Matrix: A 2D array in which the element at row \( i \) and column \( j \) indicates the presence (and possibly the weight) of an edge between vertices \( i \) and \( j \).
  • Adjacency List: An array of lists. The list at index \( i \) contains all the vertices adjacent to vertex \( i \).

Example

Consider the following undirected graph:

    A --- B
    | \   |
    |  \  |
    |   \ |
    C --- D

The adjacency matrix representation is:

       A B C D
    A [0 1 1 1]
    B [1 0 0 1]
    C [1 0 0 1]
    D [1 1 1 0]

The adjacency list representation is:

    A: B, C, D
    B: A, D
    C: A, D
    D: A, B, C

Applications

  • Networking: Graphs are used to model computer networks, social networks, and communication systems.
  • Transportation: Graphs model transportation networks such as roads, railways, and flight routes.
  • Biology: Graphs represent biological networks like neural networks and protein-protein interaction networks.
  • Operations Research: Graphs solve problems related to resource allocation, scheduling, and optimization.

Conclusion

Graph theory is a fundamental area of mathematics and computer science that provides powerful tools for modeling and solving complex problems. Understanding the basics of graph theory is essential for various applications in science, engineering, and industry.
