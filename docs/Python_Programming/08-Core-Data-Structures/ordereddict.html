OrderedDict in Python

What is an OrderedDict?

An OrderedDict is a subclass of the built-in dictionary that maintains the insertion order of keys. Unlike regular dictionaries prior to Python 3.7, OrderedDict remembers the order in which items were inserted.

Importing OrderedDict

from collections import OrderedDict

Examples

# 1. Creating an OrderedDict
from collections import OrderedDict
od = OrderedDict()
od['apple'] = 3
od['banana'] = 2
od['cherry'] = 5
print(od)  # OrderedDict([('apple', 3), ('banana', 2), ('cherry', 5)])
  
# 2. Insertion order is preserved
for key in od:
    print(key)
# Output:
# apple
# banana
# cherry
  
# 3. Updating a value does not change order
od['banana'] = 4
print(list(od.keys()))  # ['apple', 'banana', 'cherry']
  
# 4. Move a key to the end or beginning
od.move_to_end('apple')  # Moves 'apple' to the end
print(list(od.keys()))  # ['banana', 'cherry', 'apple']

od.move_to_end('banana', last=False)  # Moves 'banana' to the beginning
print(list(od.keys()))  # ['banana', 'cherry', 'apple']
  
# 5. Comparing OrderedDicts considers order
od2 = OrderedDict([('banana', 4), ('cherry', 5), ('apple', 3)])
print(od == od2)  # False because order differs
