Python Basics

Essential Concepts Before You Start Coding

1. About Python

Python is a high-level, interpreted programming language known for its readability and broad standard library. It supports multiple paradigms, including procedural, object-oriented, and functional programming.

2. Header (Shebang)

A shebang tells Unix-like systems which interpreter to use when running the script directly.

#!/usr/bin/env python3
# example.py

3. Decomposition

Break large problems into smaller, manageable functions or modules. This boosts readability, testing, and reuse.

def read_data(path):
    ...

def process(data):
    ...

def main():
    raw = read_data("data.csv")
    result = process(raw)
    print(result)

if __name__ == "__main__":
    main()

4. Interpreter vs. Compiler

Python code is executed by the Python interpreter, which converts source lines to bytecode on the fly and then runs it in the CPython virtual machine. No explicit compilation step is required.

5. Running Python

  • Interactive Mode: Start a REPL with python or python -i.
  • Script Mode: Execute a .py file: python script.py.
  • Module as Script: python -m module_name.

6. Scripts vs. Modules

A script runs top-to-bottom for its side-effects, while a module exposes functions, classes, or variables to import elsewhere. Guard reusable code with if __name__ == "__main__" so it only executes when run directly.

7. pylint & Code Quality

pylint is a static code analyzer that flags errors and enforces style conventions.

pip install pylint
pylint my_project/

8. Comments

Use comments to clarify why code exists, not what it does (which should be evident).

# Calculate compound interest over n years
amount = principal * (1 + rate) ** years

9. print()

The built-in print() function sends text to stdout. Use f-strings for readable formatting.

name = "Sai"
print(f"Hello, {name}!")

10. Indentation

Python uses indentation to define blocks; the standard is 4 spaces. Mixing tabs and spaces raises IndentationError.

for i in range(3):
    print(i)
    if i == 1:
        break

11. Line Continuation

Long statements can span multiple lines using a backslash \ or by wrapping in parentheses.

total = (price
         + tax
         - discount)
