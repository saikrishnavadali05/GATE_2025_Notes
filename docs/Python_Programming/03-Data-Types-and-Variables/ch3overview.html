1. Introduction to Python Data Types

Data types define the kind of value a variable holds. Python is dynamically typed, so it infers the data type at runtime.

2. Basic Data Types

  • int Whole numbers.
    • x = 10
  • float Decimal numbers.
    • pi = 3.14
  • bool True or False.
    • is_valid = True
  • NoneType Represents "no value".
    • value = None

3. Text Type

str String of characters.

name = "John Doe"

4. Sequence Types

  • list
    • fruits = ["apple", "banana"]
  • tuple
    • point = (1, 2)
  • range
    • for i in range(5):
    print(i)

5. Mapping Type

dict Key-value pairs.

student = {
    "name": "Alice",
    "age": 21,
    "course": "Python"
}

6. Set Types

  • set
    • colors = {"red", "green", "blue"}
  • frozenset
    • fs = frozenset([1, 2, 3])

7. Type Conversion

int("5")        # 5
str(10)         # "10"
list("abc")     # ['a', 'b', 'c']

8. Type Checking

type(10)                 # <class 'int'>
isinstance(3.14, float)  # True

9. Mutability of Data Types

Data Type Mutable? Examples
int, float, str, tuple, frozenset No Value can't change after creation
list, dict, set Yes Values can be updated

  • Convert a string "123" into an integer and add 10.
  • Create a dictionary for a student's name, ID, and course.
