Flask: Lightweight Web Framework

Flask is a popular Python microframework designed to make web development simple and flexible. It allows developers to build web applications with minimal setup, focusing on extensibility and modular design.

Key Features of Flask

  • Simple and lightweight core with a modular design
  • Easy routing and URL mapping with decorators
  • Integrated development server and debugger for rapid development
  • Template rendering with Jinja2 for dynamic HTML pages
  • Support for RESTful request dispatching
  • Session management and secure cookies
  • Extension support for databases, authentication, forms, and more
  • Flexible configuration options
  • Strong community and rich ecosystem of extensions
  • Ideal for small to medium web applications and APIs

10 Practical Flask Examples


from flask import Flask, request, jsonify, render_template, redirect, url_for, session

# Example 1: Basic Flask app
app = Flask(__name__)

@app.route('/')
def home():
    return "Hello, Flask!"

# Example 2: Route with variable
@app.route('/user/<username>')
def show_user(username):
    return f"User: {username}"

# Example 3: HTTP methods - GET and POST
@app.route('/login', methods=['GET', 'POST'])
def login():
    if request.method == 'POST':
        username = request.form['username']
        return f"Logged in as {username}"
    return '''
        <form method="post">
          Username: <input type="text" name="username">
          <input type="submit" value="Login">
        </form>
    '''

# Example 4: JSON response
@app.route('/api/data')
def api_data():
    return jsonify({"name": "Flask", "version": "2.x"})

# Example 5: Using templates (requires a 'templates' folder with index.html)
@app.route('/template')
def template():
    return render_template('index.html', title="Welcome", message="Hello from Flask")

# Example 6: Redirect and url_for
@app.route('/redirect')
def redirect_example():
    return redirect(url_for('home'))

# Example 7: Using sessions
app.secret_key = 'supersecretkey'

@app.route('/set_session')
def set_session():
    session['user'] = 'JohnDoe'
    return "Session set"

@app.route('/get_session')
def get_session():
    user = session.get('user', 'Not set')
    return f"User in session: {user}"

# Example 8: Custom error handler
@app.errorhandler(404)
def page_not_found(e):
    return "Sorry, page not found!", 404

# Example 9: Serving static files (place your files in 'static' folder)
@app.route('/static-example')
def static_example():
    return ''

# Example 10: Running the app
if __name__ == '__main__':
    app.run(debug=True)
