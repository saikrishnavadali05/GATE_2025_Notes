Sentiment Analysis in NLP

Sentiment Analysis, also called opinion mining, is the computational study of opinions, sentiments, and emotions expressed in text. It is widely used in understanding customer feedback, social media, and market research.

Types of Sentiment Analysis

  • Polarity Detection: Classifying text as positive, negative, or neutral.
  • Emotion Detection: Identifying specific emotions like joy, anger, sadness.
  • Aspect-based Sentiment Analysis: Extracting sentiment about specific aspects or features.

Popular Techniques

  • Lexicon-based approaches (e.g., VADER, TextBlob)
  • Machine learning models (SVM, Naive Bayes)
  • Deep learning models (RNN, LSTM, Transformers)
  • Pretrained pipelines from libraries like Hugging Face

10 Examples of Sentiment Analysis

from textblob import TextBlob
from vaderSentiment.vaderSentiment import SentimentIntensityAnalyzer
from transformers import pipeline

# Example 1: Simple polarity detection using TextBlob
text = "I love this product! It works wonderfully."
blob = TextBlob(text)
print(blob.sentiment.polarity)

# Example 2: Using VADER for social media text
analyzer = SentimentIntensityAnalyzer()
scores = analyzer.polarity_scores(text)
print(scores)

# Example 3: Sentiment classification with Hugging Face transformers
sentiment_pipeline = pipeline("sentiment-analysis")
result = sentiment_pipeline(text)
print(result)
