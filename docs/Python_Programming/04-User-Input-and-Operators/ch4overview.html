Python Data Types & Operators

Foundations for Writing Effective Python Code

1. Python Data Types Overview

Data types determine the kind of operations that can be applied to values. Python is dynamically typed, meaning the interpreter infers the type at runtime.

2. Primitive (Basic) Data Types

  • int Whole numbers.
    • age = 27
  • float Decimal numbers.
    • temperature = 36.6
  • bool Boolean truth values.
    • is_logged_in = False
  • NoneType Absence of a value.
    • result = None

3. Text Type

str Immutable sequence of Unicode characters.

greeting = "Hello, World!"

4. Sequence Types

  • list Mutable ordered collection.
    • cities = ["Hyderabad", "Mysuru", "Coorg"]
  • tuple Immutable ordered collection.
    • origin = (0, 0)
  • range Arithmetic progression of integers.
    • for i in range(1, 4):
    print(i)

5. Mapping Type

dict Mutable ke alue store.

employee = {
    "id": 101,
    "name": "Sai Krishna",
    "role": "Engineer"
}

6. Set Types

  • set Unordered collection of unique elements.
    • unique_numbers = {1, 2, 3}
  • frozenset Immutable version of a set.
    • constants = frozenset({3.14, 2.718})

9. Python Operators

Operators are symbols that let you perform computations and comparisons.

9.1 Arithmetic Operators

a, b = 9, 4
print(a + b)  # 13
print(a - b)  # 5
print(a * b)  # 36
print(a / b)  # 2.25
print(a // b) # 2
print(a % b)  # 1
print(a ** b) # 6561

9.2 Comparison Operators

print(a == b)  # False
print(a != b)  # True
print(a > b)   # True
print(a < b)   # False
print(a >= b)  # True
print(a <= b)  # False

9.3 Logical Operators

x, y = True, False
print(x and y)  # False
print(x or y)   # True
print(not x)    # False

9.4 Assignment Operators

counter = 10
counter += 5   # 15
counter *= 2   # 30
counter //= 4  # 7

9.5 Membership & Identity Operators

# Membership
letters = ["a", "b", "c"]
print("a" in letters)   # True
print("d" not in letters) # True

# Identity
p = [1, 2]
q = p
r = [1, 2]
print(p is q)  # True (same object)
print(p is r)  # False (same content, different objects)
