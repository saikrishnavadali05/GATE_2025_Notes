Escape Sequence: Carriage Return (\r) in Python

What is the Carriage Return Escape Sequence?

The carriage return escape sequence \\r moves the cursor back to the beginning of the current line without advancing to the next line. When printed, it can overwrite the beginning of the line with new content.

Examples of Using \\r

# Example 1: Simple overwrite
print("Hello World\rPython")

# Output:
# Python World
  
# Example 2: Overwriting entire line
print("1234567890\rABCDE")

# Output:
# ABCDE67890
  
# Example 3: Using carriage return with print progress
import time
print("Loading", end="")
for i in range(5):
    print(".", end="", flush=True)
    time.sleep(0.5)
print("\rDone!      ")

# Output:
# Done!
  
# Example 4: Carriage return with string formatting
text = "Hello\rBye"
print(text)

# Output:
# Byeo
  
# Example 5: Mixing carriage return with newline
print("First Line\rSecond\r\nThird Line")

# Output:
# Second
# Third Line

Summary

The \\r escape sequence allows overwriting of text on the current line by returning the cursor to the start of that line. It is useful in command-line interfaces to update text dynamically, such as progress indicators.
