Linear Search

Back to Programming, Data Structures, and Algorithms Topics

Linear search is a straightforward search algorithm that checks each element in a list sequentially until the target element is found or the list ends. It is also known as sequential search.

Algorithm

The linear search algorithm operates as follows:

  1. Start at the first element of the list.
  2. Compare the current element with the target element.
  3. If the current element matches the target, return the index of the current element.
  4. If the current element does not match the target, move to the next element.
  5. Repeat steps 2-4 until the target element is found or the list ends.
  6. If the target element is not found in the list, return -1.

Example

Consider the list \([4, 2, 7, 1, 9]\) and a target value of 7. The linear search algorithm proceeds as follows:

  1. Compare 4 with 7 (not a match).
  2. Compare 2 with 7 (not a match).
  3. Compare 7 with 7 (match found at index 2).

Python Implementation

def linear_search(arr, target):
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1

# Example usage
arr = [4, 2, 7, 1, 9]
target = 7
result = linear_search(arr, target)
print('Target found at index:', result)

Time Complexity

The time complexity of linear search is \( O(n) \), where \( n \) is the number of elements in the list. In the worst case, the algorithm has to check each element in the list once.

Applications

  • Small Lists: Linear search is effective for small lists where the overhead of more complex algorithms is not justified.
  • Unsorted Lists: Useful when the list is unsorted, as other more efficient search algorithms require sorted lists.
  • Finding a Specific Element: When the position of the target element is unknown and needs to be found.

Conclusion

Linear search is a simple and intuitive search algorithm that is useful for small or unsorted lists. While it is not the most efficient search algorithm for large datasets, it is easy to implement and understand, making it a valuable tool for certain applications.
