Insertion Sort

Back to Programming, Data Structures, and Algorithms Topics

Insertion sort is a simple and intuitive comparison-based sorting algorithm. It builds the sorted array one element at a time by repeatedly picking the next element and inserting it into the correct position among the previously sorted elements.

Algorithm

The insertion sort algorithm operates as follows:

  1. Start with the second element in the array (the first element is considered sorted).
  2. Compare the current element with the elements in the sorted portion of the array.
  3. Shift elements in the sorted portion to the right to create the correct position for the current element.
  4. Insert the current element into its correct position.
  5. Repeat steps 2-4 for all remaining elements in the array.

Example

Consider the array \([5, 3, 8, 4, 2]\). The insertion sort algorithm proceeds as follows:

  1. Initial array: \([5, 3, 8, 4, 2]\)
  2. Insert 3: \([3, 5, 8, 4, 2]\)
  3. Insert 8: \([3, 5, 8, 4, 2]\)
  4. Insert 4: \([3, 4, 5, 8, 2]\)
  5. Insert 2: \([2, 3, 4, 5, 8]\)

After all elements are processed, the array is sorted.

Python Implementation

def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and key < arr[j]:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key

# Example usage
arr = [5, 3, 8, 4, 2]
insertion_sort(arr)
print('Sorted array:', arr)

Time Complexity

The time complexity of insertion sort is \( O(n^2) \) in the worst and average cases, where \( n \) is the number of elements in the array. This is because each element must be compared with and potentially shifted past every other element. However, the time complexity is \( O(n) \) in the best case when the array is already sorted.

Applications

  • Small Datasets: Insertion sort is efficient for small datasets where the simplicity of the algorithm outweighs its quadratic time complexity.
  • Nearly Sorted Data: Insertion sort performs well on nearly sorted data, as it requires fewer shifts.
  • Online Sorting: Insertion sort can be used to sort data as it is received, since it processes one element at a time.

Conclusion

Insertion sort is a straightforward and easy-to-implement sorting algorithm. While it is not efficient for large datasets, it is useful for small datasets, nearly sorted data, and scenarios where simplicity and ease of implementation are more important than performance.
