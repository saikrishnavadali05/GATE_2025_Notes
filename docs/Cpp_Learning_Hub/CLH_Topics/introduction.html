What is C++?

C++ is a general-purpose programming language created by Bjarne Stroustrup in 1985. It is widely used for system/software development, game development, and competitive programming.

Features of C++

  • Object-Oriented Programming
  • High Performance
  • Standard Template Library (STL)
  • Portability
