Cumulative Distribution Function (CDF)

Back to Probability and Statistics Topics

The cumulative distribution function (CDF) is a fundamental concept in probability theory and statistics. It describes the probability that a random variable takes on a value less than or equal to a given point. The CDF provides a complete description of the distribution of a random variable.

Definition

For a random variable \(X\), the cumulative distribution function \(F(x)\) is defined as:

\[ F(x) = P(X \leq x) \]

This function gives the probability that the random variable \(X\) will take a value less than or equal to \(x\).

Properties of the CDF

  • Non-decreasing: The CDF is a non-decreasing function, meaning that \(F(x_1) \leq F(x_2)\) for \(x_1 < x_2\).
  • Limits: The CDF approaches 0 as \(x\) approaches negative infinity and 1 as \(x\) approaches positive infinity:
    • \(\lim_{{x \to -\infty}} F(x) = 0\)
    • \(\lim_{{x \to \infty}} F(x) = 1\)
  • Right-continuous: The CDF is right-continuous, meaning that for any \(x\), \(\lim_{{h \to 0^+}} F(x + h) = F(x)\).

Example: Discrete Random Variable

Consider a discrete random variable \(X\) that can take on values 1, 2, or 3 with probabilities 0.2, 0.5, and 0.3, respectively. The CDF of \(X\) is given by:

F(x) = 0 for x < 1
F(x) = 0.2 for 1 ≤ x < 2
F(x) = 0.7 for 2 ≤ x < 3
F(x) = 1 for x ≥ 3

Example: Continuous Random Variable

Consider a continuous random variable \(X\) with a probability density function (PDF) \(f(x) = \frac{1}{2} e^{-|x|}\). The CDF of \(X\) is given by:

\[ F(x) = \begin{cases} \frac{1}{2} e^x & x < 0 \\ 1 - \frac{1}{2} e^{-x} & x \geq 0 \end{cases} \]

Relationship Between PDF and CDF

For a continuous random variable, the CDF is the integral of the probability density function (PDF). Conversely, the PDF is the derivative of the CDF. Mathematically, if \(X\) is a continuous random variable with PDF \(f(x)\) and CDF \(F(x)\), then:

\[ F(x) = \int_{-\infty}^{x} f(t) \, dt \]

\[ f(x) = \frac{d}{dx} F(x) \]

Applications

  • Probability Calculations: The CDF is used to calculate probabilities for intervals and specific values of the random variable.
  • Quantile Calculation: The CDF is used to determine the quantiles of a distribution, such as the median or percentiles.
  • Statistical Inference: The CDF is used in hypothesis testing and confidence interval estimation.
  • Data Analysis: Empirical CDFs are used to summarize and visualize the distribution of sample data.

Conclusion

The cumulative distribution function (CDF) is a crucial tool in probability and statistics, providing a complete description of the distribution of a random variable. By understanding and utilizing the CDF, statisticians and analysts can perform a wide range of probability calculations and statistical analyses.
