GitHub Assignment Submission Guide

This guide helps you learn how to fork, clone, push and submit assignments through GitHub pull requests.

Workflow Diagram

graph TD A[Create GitHub Account] --> B[Install Git & VS Code] B --> C[Fork Trainer's Repo] C --> D[Clone to Local Machine] D --> E[Create Branch (Optional)] E --> F[Edit/Add Assignment Files] F --> G[Commit Changes] G --> H[Push to GitHub] H --> I[Create Pull Request] I --> J[Trainer Reviews & Merges]

Prerequisites

  • Create GitHub account https://github.com
  • Install Git on your PC https://git-scm.com
  • Install Visual Studio Code on your PC https://code.visualstudio.com

Step 1: Fork the Repository

  1. Open: SSSSO-RR-District-Skill-Development-Training
  2. Click Fork on the top-right corner.
  3. You will now have your own copy.

Step 2: Clone the Fork

git clone https://github.com/YOUR-USERNAME/SSSSO-RR-District-Skill-Development-Training.git
cd SSSSO-RR-District-Skill-Development-Training

Step 3: Create a Branch (Optional)

git checkout -b your-name-assignment1

Step 4: Add Your Assignment

Edit or add files in the project folder. Save your work.

Step 5: Commit Your Work

git add .
git commit -m "Add assignment 1 solution"

Step 6: Push to GitHub

git push origin your-name-assignment1

Step 7: Create Pull Request

  1. Go to your forked repo on GitHub.
  2. Click Compare & pull request.
  3. Add a meaningful title and description.
  4. Click Create pull request.

Bonus: Sync Your Fork

git remote add upstream https://github.com/saikrishnavadali05/SSSSO-RR-District-Skill-Development-Training.git
git fetch upstream
git checkout main
git merge upstream/main
git push origin main
